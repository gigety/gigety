## Gigety 
Gigety is a project for practice and demonstration purposes of many technologies.  

### Technologies
* Docker / Docker Swarm
* Spring Boot
* Spring Security
* Spring Data
* Spring Web / Rest
* NodeJS
* Socket.io
* Oauth2 / JWT
* MongoDB
* MySQL
* Redis Cache Server
* ReactJS
* React Native
